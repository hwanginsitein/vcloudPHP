		setSortDesc (line 151)
	
 
	
	

Sets the field(s) name by which to sort in the descending order. Field  name comes first has higher priority in the sorting. For example, when  $fields = 'LastName, FirstName', the records of the query result will be  sorted first by the 'LastName' column, and then the 'FirstName' column.


This parameter can only be used when the query output format is set as  'records'.  For example: $params->setSortDesc('name,category');
