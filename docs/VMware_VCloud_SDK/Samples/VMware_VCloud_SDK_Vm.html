Get the status of this VM.

The following are the possible VM status:

  • -1 -- could not be created
  • 0 -- unresolved
  • 1 -- resolved
  • 3 -- suspended
  • 4 -- powered on
  • 5 -- waiting for user input
  • 6 -- in an unknown state
  • 7 -- in an unrecognized state
  • 8 -- powered off
