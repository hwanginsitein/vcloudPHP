Class Description
VMware_VCloud_API_ResourceReferenceType Represents a reference to a resource. Contains an href attribute, a resource status attribute, and optional name and type attributes.
VMware_VCloud_API_LinkType Extends reference type by adding relation attribute. Defines a hyper-link with a relationship, hyper-link reference, and an optional MIME type.
VMware_VCloud_API_OrganizationReferenceType Represents a reference to an organization.
